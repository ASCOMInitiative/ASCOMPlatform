Dome.Connected Property

   

 Dome.Connected  (read-write, Boolean)
 
True if driver has established communication to dome control.

Syntax

Dome.Connected [= Boolean]

The property syntax has these parts:

Part Description
Value (Boolean) True if driver has established communication to dome control.

Remarks

Set to True to establish the link and set to False to terminate the link. Raises an error if connect fails.
