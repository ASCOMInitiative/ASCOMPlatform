Rotator.MoveAbsolute() Method

   

Changes the rotator position angle to the given angular position.

Syntax

Rotator.MoveAbsolute(Position)

The method syntax has these parts:

Part Description
Position (Single) The position angle in degrees to which the rotator will be moved
Return (Nothing) Does not return a value.

Remarks

Calling MoveAbsolute() causes the TargetPosition property to change to the value of the Position parameter, then starts rotation to TargetPosition.
