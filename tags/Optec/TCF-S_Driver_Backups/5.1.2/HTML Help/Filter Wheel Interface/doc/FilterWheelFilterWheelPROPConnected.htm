FilterWheel.Connected Property

   

 FilterWheel.Connected  (read-write, Boolean)
 
State of the connection to the filter wheel

Syntax

FilterWheel.Connected [= Boolean]

The property syntax has these parts:

Part Description
Value (Boolean) State of the connection to the filter wheel

Remarks

Controls the link between the driver and the FilterWheel. Set True to enable the link. Set False to disable the link. You can also read the property to check whether it is connected.

Must throw exception if unsuccessful.
