Serial.ReceiveByte() Method

   

Receive one binary byte from the serial port

Syntax

Serial.ReceiveByte()

The method syntax has these parts:

Part Description
Return (Byte) The received byte

Remarks

Use this for 8-bit (binary data). If a timeout occurs, an automation error is raised.
