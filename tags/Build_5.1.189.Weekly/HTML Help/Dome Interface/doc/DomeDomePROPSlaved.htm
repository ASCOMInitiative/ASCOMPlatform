Dome.Slaved Property

   

 Dome.Slaved  (read-write, Boolean)
 
True if the dome is slaved to the telescope in its hardware, else False.

Syntax

Dome.Slaved [= Boolean]

The property syntax has these parts:

Part Description
Value (Boolean) True if the dome is slaved to the telescope in its hardware, else False.

Remarks

Set this property to True to enable dome-telescope hardware slaving, if supported (see Dome.CanSlave). Raises an exception on any attempt to set this property if hardware slaving is not supported). Always returns False if hardware slaving is not supported.
