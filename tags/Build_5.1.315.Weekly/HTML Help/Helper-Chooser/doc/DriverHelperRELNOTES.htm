DriverHelper Release Notes

This is the release of the ASCOM Helper component for the Platform 5.0 release. The only change from the previous release (4.1.3) is the addition of a new property SerialTimeoutMs, allowing the serial timeout to be specified in milliseconds.
Please conduct discussions and report bugs that you find in this component on the ASCOM-Talk List.
