Telescope.SlewSettleTime Property

       

 Telescope.SlewSettleTime  (read-write, Integer)
 
Specifies a post-slew settling time (sec.).

Syntax

Telescope.SlewSettleTime [= Integer]

The property syntax has these parts:

Part Description
Value (Integer) Specifies a post-slew settling time (sec.).

Remarks

Adds additional time to slew operations. Synchronous slewing methods will not return, and the Telescope.Slewing property will not become false, until the slew completes and the SlewSettleTime has elapsed. This feature (if supported) may be used with mounts that require extra settling time after a slew.
