Telescope.SyncToCoordinates() Method

       

Matches the scope's coordinates to the given coordinates.

Syntax

Telescope.SyncToCoordinates(RightAscension, Declination)

The method syntax has these parts:

Part Description
RightAscension (Double) The corrected right ascension (local topocentric, hours). Copied to Telescope.TargetRightAscension.
Declination (Double) The corrected declination (local topocentric, degrees, positive North). Copied to Telescope.TargetDeclination.
Return (Nothing) Does not return a value.

Remarks

Sets Telescope.TargetRightAscension to the given right ascension, and Telescope.TargetDeclination to the given declination. Raises an error if matching fails.
