Focuser Release Notes
V1.0 (release, rev. 24-Aug-2007 clarification only)

This is the 1.0 version of the ASCOM Focuser Interface, the first release as an ASCOM standard. For more information see the ASCOM Initiative web site. The 24-Aug-2007 revision clarifies the meaning of the Position parameter in the Focuser.Move() method. There are no changes to the interface or behavior resulting from this change.
