Profile.Unregister() Method

   

Remove all data for the given DriverID from the registry. This deletes the entire tree, including the DriverID root key.

Syntax

Profile.Unregister(DriverID)

The method syntax has these parts:

Part Description
DriverID (String) The ASCOM DriverID for the driver (also known as the automation ProgID)
Return (Nothing) Does not return a value.

Remarks

See the Profile object home page for more information on driver registration and system registry use.
