Serial.ReceiveTerminated() Method

   

Teceive text until the given terminator string is seen

Syntax

Serial.ReceiveTerminated(Terminator)

The method syntax has these parts:

Part Description
Terminator (String) The terminator string (may be more than one character)
Return (String) The received text, including the terminator

Remarks

If a timeout occurs, an automation error is raised.
