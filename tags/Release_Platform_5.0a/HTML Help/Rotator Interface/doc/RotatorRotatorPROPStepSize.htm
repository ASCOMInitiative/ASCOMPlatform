Rotator.StepSize Property

   

 Rotator.StepSize  (read-only, Single)
 
The angular size of each step in position angle, degrees.

Syntax

Rotator.StepSize

The property syntax has these parts:

Part Description
Value (Single) The angular size of each step in position angle, degrees.

Remarks

Raises an exception if the rotator does not intrinsically know what the step size is.
