Focuser.TempCompAvailable Property

   

 Focuser.TempCompAvailable  (read-only, Boolean)
 
True if focuser has temperature compensation available.

Syntax

Focuser.TempCompAvailable

The property syntax has these parts:

Part Description
Value (Boolean) True if focuser has temperature compensation available.

Remarks

Will be True only if the focuser's temperature compensation can be turned on and off via the TempComp property.
