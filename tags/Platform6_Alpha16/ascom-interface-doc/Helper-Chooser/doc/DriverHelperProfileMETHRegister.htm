Profile.Register() Method

   

Register the driver with the given DriverID and descriptive name. Does nothing if already registered, so safe to call on driver load.

Syntax

Profile.Register(DriverID, DescriptiveName)

The method syntax has these parts:

Part Description
DriverID (String) The ASCOM DriverID for the driver (also known as the automation ProgID)
DescriptiveName (String) The driver "friendly name" that will show in the Chooser's selector box
Return (Nothing) Does not return a value.

Remarks

The descriptive name shows in the Chooser's selector box. See the Profile object home page for more information on driver registration and system registry use.
