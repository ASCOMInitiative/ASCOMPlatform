Util.HoursToHMS() Method

   

Convert hours to sexagesimal hours, minutes, and seconds

Syntax

Util.HoursToHMS(Hours, [HrsDelim], [MinDelim], [SecDelim], [SecDecimalDigits])

The method syntax has these parts:

Part Description
Hours (Double) The hours value to convert
HrsDelim (String) The delimiter string separating hours and minutes (optional, default = ":")
MinDelim (String) The delimiter string separating minutes and seconds (optional, default = ":")
SecDelim (String) The delimiter string to append to the seconds part (optional, default = nothing)
SecDecimalDigits (Integer) The number of digits after the decimal point on the seconds part (optional, default = 0)
Return (String) Sexagesimal representation of hours input value, hours, minutes and seconds

Remarks

If you need a leading plus sign, you must prepend it yourself. The delimiters are not restricted to single characters.

The default representation is HH:MM:SS.
