Serial Object

               

ASCOM Scope Driver Helper Serial Port Object

Remarks

This object provides an easy to use interface to a serial (COM) port. It provides ASCII and binary I/O with controllable timeout. It is limited to 8 bit, no parity, one stop-bit modes. The interface is callable from any automation-compliant client including a Windows Script Component (script implementation of an object, see the Windows Script Host documentation that comes with the ASCOM Platform). The object implements IObjectSafety, and reports itself safe for scripting, thus it may be used from within a web browser and dynamic HTML as well.
Calls into the Serial object are guaranteed not to pump messages while waiting for completion of requested operations. Thus, you are safe from being re-entered on the thread that called Serial.
