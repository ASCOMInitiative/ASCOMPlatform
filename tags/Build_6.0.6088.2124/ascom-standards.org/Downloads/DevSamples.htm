Developer Samples & Sources

The ASCOM Platform 6 contains developer templates for Visual Studio 2008 and all types of drivers in both C#.NET and VisualBasic.NET. If you're developing a driver using VS.NET, it is strongly recommended that you use the relevant template. The Platform also contains an interface toolkit for astronomy software developers who want to use drivers from .NET.

Using the ASCOM .NET Client Toolkit from C#
A Visual Studio C# project that shows how to use the ASCOM .NET Client Toolkit. This is a console application which contains code that access each type of driver, using the chooser and pre-selecting the simulator for each. Some info is printed out to the console window.
