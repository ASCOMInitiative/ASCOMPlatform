Profile.DeleteValue() Method

   

Delete the value from the registry. Name may be an empty string for the unnamed value. Optional path to subkey for given Driver ID may be supplied.

Syntax

Profile.DeleteValue(DriverID, Name, [SubKey])

The method syntax has these parts:

Part Description
DriverID (String) The ASCOM DriverID for the driver (also known as the automation ProgID)
Name (String) The name of the value to be deleted. Specify "" to delete the unnamed value.
SubKey (String) Name of sub-key under which the value to be deleted is located. May be a \ separated path. (optional, default = "")
Return (Nothing) Does not return a value.

Remarks

The unnamed value is also known as the "default" value for a registry key.
