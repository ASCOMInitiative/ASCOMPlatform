Optec Intelligent Filter Wheel Driver

You have just successfully installed an ASCOM compliant driver for Optec Inc's Intellegent Filter Wheel devices (IFW)

Driver Version:

  • You have installed Version 1.1.0.1 for the Optec IFW

Description:

  • This driver is compatable with Optecs IFW AND IFW-3
  • The methods in this driver are 100% compliant with the standards set in place by the ASCOM open source group. Any software which is capable of interfacing with ASCOM standard filterwheels may use this driver.
  • Click here for details regarding the ASCOM FilterWheel driver interface

Driver Setup Details:

  • Before using your IFW or IFW-3 you MUST enter the driver properties and complete the initial device setup. From the Driver Chooser simply click on the Properties button to enter the setup.
  • The following basic settings can be configured in the driver properties window

    • Select the COM port to which your Filter Wheel is connected
    • Select the type (IFW or IFW-3) of Filter Wheel you have connected
    • Enter descriptive names for each filter in your wheel (example IR-10)
  • By clicking the Advanced button in the driver setup form you can access the folowing tools:

    • Next and Back Centering Constants - This allows you to adjust where the filter wheel stops after the filter wheel has changed positions.
    • The Serial Com Debug interface allows you to manually send commands to the IFW. The Command Reference box lists the syntax of the supported commands. This interface can be used to verify proper functionality of the IFW.
