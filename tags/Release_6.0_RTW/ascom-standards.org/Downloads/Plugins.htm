Plug-Ins

Mount/Telescope X2 to ASCOM Converter for TheSky X (BETA)
Plug-in that allows Software Bisque's TheSky X to control telescopes via ASCOM telescope drivers. Requires TheSky X Professional Edition for WIndows Daily Build (May 16). For more information see Working with TheSky (section 2). This is a self-contained installer. Thanks to Bob Denny of DC-3 Dreams for this.

TheSky™ ASCOM Telescope (TeleAPI) Plug-In for TheSky 5/6 (5.0.4)
Use the X2 plug-ins above for TheSky X. Plug-in that allows Software Bisque's TheSky X/6/5 Astronomy Software to control telescopes via ASCOM telescope drivers. For more information see Working with TheSky (section 3). This is a self-contained installer.
CCDSoft™ ASCOM Camera Plug-In
Plug-in that allows Software Bisque's CCDSoft Imaging Software to control cameras via their ASCOM drivers. Useful for one shot color and other cameras not natively supported. The plug-in is contained in its own installer. Thanks to Evan Warkentine of ShrimpBox Astronomy for this.
AutomaDome™ ASCOM Dome (DomeAPI) Plug-In
Plug-in that allows Software Bisque's AutomaDome Dome Control Software to control domes via ASCOM dome drivers. The plug-in is contained in its own installer.
CCDSoft™ ASCOM Focuser (FocusAPI) Plug-In
Plug-in that allows Software Bisque's CCDSoft Imaging Software to control focusers via ASCOM dome drivers. The plug-in is contained in its own installer.
SkyMap Pro ASCOM Plug-In
Plug-in that allows Chris Marriott's SkyMap Pro 11 planetarium and star-charting software to control telescopes via ASCOM telescope drivers. This is a zip file with a read-me document describing installation and usage.
