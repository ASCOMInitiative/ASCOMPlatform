Serial.Speed Property

   

 Serial.Speed  (read-write, PortSpeed)
 
The serial communication data speed to use (baud, default = 9600)

Syntax

Serial.Speed [= PortSpeed]

The property syntax has these parts:

Part Description
Value (PortSpeed) The serial communication data speed to use (baud, default = 9600)

Remarks
