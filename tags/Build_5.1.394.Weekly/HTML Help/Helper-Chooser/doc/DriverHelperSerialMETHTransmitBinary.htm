Serial.TransmitBinary() Method

   

Transmit an array of binary bytes

Syntax

Serial.TransmitBinary(Data)

The method syntax has these parts:

Part Description
Data (Variant) Must be an automation-type array of Byte data.
Return (Nothing) Does not return a value.

Remarks

Automation arrays are sometimes known as SAFEARRAY types. The array must contain elements if type VARIANT with the variant type unsigned 8 bit (Byte() in Visual Basic).
