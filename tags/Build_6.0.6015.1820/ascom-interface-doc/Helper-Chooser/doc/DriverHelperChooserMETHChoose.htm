Chooser.Choose() Method

   

Select ASCOM Telescope Driver to use, return Driver ID of chosen driver

Syntax

Chooser.Choose([CurrentDriverID])

The method syntax has these parts:

Part Description
CurrentDriverID (String) The caller's currently selected DriverID (optional, default=nothing)
Return (String) The DriverID of the chosen driver, or "" (empty string) if nothing chosen or chooser Cancel button was clicked

Remarks

If you supply a CurrentDriverID, the corresponding driver will be pre-selected in the Chooser's list when the choosec window is first opened. The parameter is optional, and if omitted or passed "", no driver will be pre-selected in the chooser window.
