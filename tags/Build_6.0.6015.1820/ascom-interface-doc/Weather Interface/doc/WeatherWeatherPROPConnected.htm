Weather.Connected Property

   

 Weather.Connected  (read-write, Boolean)
 
True if the server has established communication with the weather data provider

Syntax

Weather.Connected [= Boolean]

The property syntax has these parts:

Part Description
Value (Boolean) True if the server has established communication with the weather data provider

Remarks

Set to True to establish communications with the weather server's specific weather data provider. Set to False to terminate the link. It is permitted for this property to be read-only for the case where the server and the weather data provider are integrated, and thus there is no need to connect and disconnect.

Raises an error if Connect is set to True and the connection fails. Also raises an error Connected is read-only and an attempt is made to change its value.
