Introduction

The ASCOM Rotator Interface allows instrument rotators from a variety of manufacturers to be connected in a standard way with different software applications. It provides an interface which is uniform and consistent, regardless of the actual hardware used.

The driver supports both absolute and relative rotator operation. Any special features of a rotator that require special setup can be accommodated using a Setup dialog box which can be opened by the host software.

Scope

This specification applies only to rotators used on equatorial mounts for positioning the imager relative to the sky. It does not apply to "de-rotators" used on alt-az mounts.
