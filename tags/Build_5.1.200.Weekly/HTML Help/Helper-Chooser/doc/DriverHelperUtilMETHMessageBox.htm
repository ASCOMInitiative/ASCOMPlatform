Util.MessageBox() Method

   

Display an alert box with configurable icon, text, caption, and button set. Return an answer appropriate to selected button set.

Syntax

Util.MessageBox(Message, [Options], [title])

The method syntax has these parts:

Part Description
Message (String) The text of the message to display in the alert box.
Options (MessageBoxOptions) One or more options (added together) to control icon, button set, and modality. See table below. (optional)
title (String) Text to appear in the title bar of the alert box (optional, default = "DriverHelper.Util Object")
Return (MessageBoxAnswers) Indication of button pressed to close the alert box. See table below.

Remarks

Provides Win32 MessageBox() functionality for scripting languages that do not have this capability (e.g., JScript). The Options parameter consists of one or more MessageBoxOption values added together. For example, to display a Yes/No confirmation alert with a question mark icon 
ans = Util.MessageBox("Are you happy?", _
    (mbQuestion + mbYesNo), "Answer Truthfully"
ans will contain vbYes or vbNo (one of the MessageBoxAnswer constants).

Symbolic Constants

The (symbolic) values for MessageBoxAnswers are:

mbOK 1 mbOK
mbCancel 2 mbCancel
mbAbort 3 mbAbort
mbRetry 4 mbRetry
mbIgnore 5 mbIgnore
mbYes 6 mbYes
mbNo 7 mbNo
