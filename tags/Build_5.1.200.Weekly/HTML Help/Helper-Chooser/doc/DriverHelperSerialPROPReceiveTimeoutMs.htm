Serial.ReceiveTimeoutMs Property

   

 Serial.ReceiveTimeoutMs  (read-write, Long)
 
The maximum time that the object will wait for incoming receive data (milliseconds, default = 5000)

Syntax

Serial.ReceiveTimeoutMs [= Long]

The property syntax has these parts:

Part Description
Value (Long) The maximum time that the object will wait for incoming receive data (milliseconds, default = 5000)

Remarks

If a timeout occurs, an automaton error is raised. See ReceiveTimeout for an alternate form using the timeout in seconds.
