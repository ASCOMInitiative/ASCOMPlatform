Util.TimeZoneOffset Property

   

 Util.TimeZoneOffset  (read-only, Double)
 
UTC offset (hours) for the computer's clock

Syntax

Util.TimeZoneOffset

The property syntax has these parts:

Part Description
Value (Double) UTC offset (hours) for the computer's clock

Remarks

The offset is in hours, such that UTC = local + offset. The offset includes any daylight/summer time that may be in effect.
