Dome.CommandString() Method

   

Send a string command directly to the dome, returning the response string.

Syntax

Dome.CommandString(Command)

The method syntax has these parts:

Part Description
Command (String) Raw command string to be sent to the dome.
Return (String) Response string from controller.

Remarks

Raises an error if not supported or if a communications failure occurs.
