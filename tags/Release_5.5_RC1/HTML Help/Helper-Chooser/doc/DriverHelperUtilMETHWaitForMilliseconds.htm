Util.WaitForMilliseconds() Method

   

Pauses for a given interval in milliseconds.

Syntax

Util.WaitForMilliseconds(Milliseconds)

The method syntax has these parts:

Part Description
Milliseconds (Long) The length of time to pause (milliseconds)
Return (Nothing) Does not return a value.

Remarks

Repeatedly puts the calling Win32 process to sleep, totally freezing it, for 100 milliseconds, then pumps events so the script or program calling it will receive its normal flow of events, until the pause interval elapses. If the pause interval is 1000 milliseconds or less, the sleep interval is reduced to 0, causing the calling Win32 process to give up control to the kernel scheduler and then immediately become eligible for scheduling.
