Rotator.Connected Property

   

 Rotator.Connected  (read-write, Boolean)
 
True if driver has established communications to the rototor

Syntax

Rotator.Connected [= Boolean]

The property syntax has these parts:

Part Description
Value (Boolean) True if driver has established communications to the rototor

Remarks

Set to True to establish the link and set to False to terminate the link. The current Connected status can also be read back as this property. Raises an error if connect fails.

A call to set Connected to True must not return until the hardware and driver are ready for use. It is up to the driver and hardware to perform any needed power-up initialization (such as homing/indexing or wrap management) before returning. When Connect returns, the rotator must be ready to use.
