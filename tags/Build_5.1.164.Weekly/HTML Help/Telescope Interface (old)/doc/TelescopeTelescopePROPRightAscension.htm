Telescope.RightAscension Property

       

 Telescope.RightAscension  (read-only, Double)
 
The right ascension (hours) of the telescope's current local topocentric equatorial coordinates

Syntax

Telescope.RightAscension

The property syntax has these parts:

Part Description
Value (Double) The right ascension (hours) of the telescope's current local topocentric equatorial coordinates

Remarks

Reading the property will raise an error if the value is unavailable.
