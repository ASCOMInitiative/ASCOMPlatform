Timer.Enabled Property

   

 Timer.Enabled  (read-write, Boolean)
 
True means the timer is active and will deliver Tick events every Interval milliseconds (default = False)

Syntax

Timer.Enabled [= Boolean]

The property syntax has these parts:

Part Description
Value (Boolean) True means the timer is active and will deliver Tick events every Interval milliseconds (default = False)
