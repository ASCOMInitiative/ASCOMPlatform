Util.LoadDLL() Method

   

Load a dynamic link library

Syntax

Util.LoadDLL(FileName)

The method syntax has these parts:

Part Description
FileName (String) The full path/name of the DLL to be loaded.
Return (Nothing) Does not return a value.

Remarks

You can get the path to your driver's installation directory in Visual Basic as App.Path. Thus to install a DLL located in your driver's installation directory: 
Profile.LoadDLL App.Path & "\MyLib.dll"
