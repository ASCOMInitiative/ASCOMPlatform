Profile.DeleteSubKey() Method

   

Delete a registry key for the given DriverID. SubKey may contain \ separated path to key to be deleted.

Syntax

Profile.DeleteSubKey(DriverID, SubKey)

The method syntax has these parts:

Part Description
DriverID (String) The ASCOM DriverID for the driver (also known as the automation ProgID)
SubKey (String) The name of the sub-key to create. May contain \ separated path.
Return (Nothing) Does not return a value.

Remarks

The sub-key and all data and keys beneath it are deleted.
