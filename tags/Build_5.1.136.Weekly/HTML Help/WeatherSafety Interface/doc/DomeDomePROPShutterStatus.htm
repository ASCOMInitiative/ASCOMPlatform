Dome.ShutterStatus Property

   

 Dome.ShutterStatus  (read-only, ShutterState)
 
Ttatus of the dome shutter or roll-off roof.

Syntax

Dome.ShutterStatus

The property syntax has these parts:

Part Description
Value (ShutterState) Ttatus of the dome shutter or roll-off roof.

Remarks

Raises an error only if no shutter control. If actual shutter status can not be read, then reports back the last shutter state.

Symbolic Constants

The (symbolic) values for ShutterState are:

Constant Value Description
shutterOpen 0 shutterOpen
shutterClosed 1 shutterClosed
shutterOpening 2 shutterOpening
shutterClosing 3 shutterClosing
shutterError 4 shutterError
