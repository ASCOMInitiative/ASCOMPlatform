Profile.SubKeys() Method

   

Return a list of the sub-keys under the given DriverID. Optional path to starting sub-key for given Driver ID may be given.

Syntax

Profile.SubKeys(DriverID, [SubKey])

The method syntax has these parts:

Part Description
DriverID (String) The ASCOM DriverID for the driver (also known as the automation ProgID)
SubKey (String) Name of sub-key under which the sub-keys below are to be listed. May be a \ separated path. (optional, default = "")
Return (Object) A Scripting.Dictionary object containing entries for each sub-key.

Remarks

The returned Scripting.Dictionary object contains entries for each sub-key. For each entry in the Dictionary, the Key property is the sub-key name, and the Value property is the unnamed ("default") string value for that key.

Calling 

Profile.SubKeys("")
will return a Dictionary containing the currently registered ASCOM Telescope Drivers.
