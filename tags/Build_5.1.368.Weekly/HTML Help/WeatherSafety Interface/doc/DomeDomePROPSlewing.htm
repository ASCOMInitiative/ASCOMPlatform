Dome.Slewing Property

   

 Dome.Slewing  (read-only, Boolean)
 
True if any part of the dome is currently moving, False if all dome components are steady.

Syntax

Dome.Slewing

The property syntax has these parts:

Part Description
Value (Boolean) True if any part of the dome is currently moving, False if all dome components are steady.

Remarks

Raises an error if a communications error occurs.
