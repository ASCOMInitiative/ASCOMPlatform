Installing the ASCOM Platform on Windows 7

The ASCOM Platform depends on Microsoft technology that is included in Windows 7, but is not enabled by default. Here's what you need to do before installing the ASCOM Platform on Windows 7:

  1. Open the Control Panel
  2. Bring up the Programs and Features applet
  3. Click on the link "Turn Windows features on or off"
  4. Put a checkmark next to Microsoft .NET Framework 3.5.1
  5. Click OK and close out to Windows (this will enable the .NET Framework as required)
  6. Now install the ASCOM Platform
