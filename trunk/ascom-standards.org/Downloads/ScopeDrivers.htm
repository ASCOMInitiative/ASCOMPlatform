Telescope/Mount Drivers

ACL Telescope Controllers (5.0.1)
Installs a driver for telescope control systems that speak the Merlin Controls Astronomy Command Language. Please note that there is a specific driver for the COMSOFT PC-TCS control system below.
Astro-Physics GTO (5.0.0, combined telescope/focuser driver)
This is the installer for the Astro-Physics factory-supported driver for all Astro-Physics telescope models. The previous 4.x versions of this driver should no longer be used.
Celestron Unified (5.0.20)
Installs a driver for all computerized Celestron telescopes. This version has passed the Conformance Checker tests (see this report) and has support for programmably changing pier side, the CGE Pro and CGEM models (with Celestron Guide commands). Works for Locales where the comma is the decimal separator.
EQMOD Project (external link)
With the EQMOD ASCOM driver, the user will be able to control the Modded EQ mount using any ASCOM based astronomy software at 0.144 arcsecond resolution. This is an ongoing project, so be sure to check the site for updates.
Gemini Telescope .NET (1.0.8.0)
Successor to Gemini Controller, installs a driver for mounts controlled by René Görlich's Gemini Astronomical Telescope Positioning System. Current firmware required! This version has passed the Conformance Checker tests (see this report)
Generic LX-200 (5.0.1)
Installs a driver that uses a small subset of the LX-200 protocol. Features are limited, and end-of-slew detection may be unreliable (requires adjustment of encoder resolution). Use this driver with mounts that purport to use the LX-200 protocol. Be prepared for limitations and problems. Simple applications (such as planetarium slewing) are more likely to work than demanding automation applications.
Intelliscope (version unknown, for Platform 5)
Driver for the Orion Intelliscope telescope control system as well as many Digital Setting Circles controllers. the author provides a list of compatible DSCs and a self-insteller for the driver.
LittleFoot Controller Project (1.0.0)
Driver for the LittleFoot Controller Project, specifically the LittleFoot Classic. For more info see Astronomie & Elektronik (a German/English forum).
Meade LX200 Classic and Autostar #494, #495, and #497 (combined telescope/focuser, 5.0.3)
Installs a driver for the original Meade LX200 ("classic") telescope mounts (including the 16"), as well as Meade telescopes that use the Autostar I (#494, #495 and #497) controllers, such as the ETX and DS series. This driver provides both telescope and focuser interfaces each of which may be used by separate programs. It also has support for an improved form of PulseGuide. Do not use this with the LX200GPS, LX200R or the RCX400. V5.0.3 corrects an installer error.
Meade LX200GPS and LX200R (combined telescope/focuser, 5.0.0)
Installs a driver for the Meade LX200GPS and LX200R telescope mounts. This driver provides both telescope and focuser interfaces each of which may be used by separate programs. Do not use this with the LX200 Classic or the Autostar I (#495/#497) controllers. It has been reported that this driver also works with the RCX400 except it doesn't allow access to its unique features.
PC-TCS (combined telescope/focuser/dome 5.0.1)
Installs a driver for the COMSOFT PC-TCS telescope control systems. This driver provides combined telescope, focuser, and dome interfaces each of which may be used by separate programs.
ServoCAT (4.2.2)
Driver for the StellarCAT ServoCAT telescope control system. The author provides a self-contained installer.
Takahashi Temma (5.0.26, external link)
Driver for the Takahashi Temma mounts. This is a full-featured driver that provides not only ASCOM standard driver functions, but also user access to many of the Temma's special features.
TheSky Controlled Telescope (Paramount, etc., 5.2.4) for TheSky X and TheSky 5/6
Installs a driver that connects to Software Bisque's TheSky X/6/5 Astronomy Software and makes it look like a standard ASCOM telescope. This allows ASCOM-based astronomy software to use TheSky (including TPOINT and ProTrack) as a smart telescope controller. For more information see Working with TheSky (section 1). It is required for the Paramount mounts, and may be used with TheSky controlling any other of its supported telescope types, or via the ASCOM plugin, using ASCOM telescope drivers. Version 5.2.4 adds full Telescope V2 support, 100% conformance, and compatibility with TheSky X (10.1.8 or later). The driver is licensed per Creative Commons.
Vixen SkySensor 2000 (5.1.7b)
Installs a driver for the Vixen SkySensor 2000 telescope control system. This version has ASCOM 5 early binding and has passed the Conformance Checker tests (see this report).
Vixen Sphinx (external link)
Driver for the Vixen Sphinx mounts.
