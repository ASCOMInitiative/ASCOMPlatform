Plug-Ins

TheSky™ ASCOM Telescope (TeleAPI) Plug-In for TheSky X and TheSky 5/6 (5.0.3)
Plug-in that allows Software Bisque's TheSky X/6/5 Astronomy Software to control telescopes via ASCOM telescope drivers. For more information see Working with TheSky (section 2). This is a self-contained installer. Please read the document that appears at the end of installation! 5.0.3 adds the special registry data needed for TPOINT and German Equatorial mounts (untested on TheSky X).
AutomaDome™ ASCOM Dome (DomeAPI) Plug-In
Plug-in that allows Software Bisque's AutomaDome Dome Control Software to control domes via ASCOM dome drivers. The plug-in is contained in its own installer.
CCDSoft™ ASCOM Focuser (FocusAPI) Plug-In
Plug-in that allows Software Bisque's CCDSoft Camera Control Software to control focusers via ASCOM dome drivers. The plug-in is contained in its own installer.
Starry Night 3/4 Telescope Control Plug-in
Plug-in that allows the obsolete Starry Night Pro versions 3 and 4 to control telescopes via ASCOM telescope drivers. Do not use this with Starry Night Pro V5 or V6. Contact Imaginova for the plug-ins for these versions. This is a zip file with a read-me document describing installation and usage.
SkyMap Pro ASCOM Plug-In
Plug-in that allows Chris Marriott's SkyMap Pro 11 planetarium and star-charting software to control telescopes via ASCOM telescope drivers. This is a zip file with a read-me document describing installation and usage.
