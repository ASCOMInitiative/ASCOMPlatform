Plug-Ins

Most of these plug-ins provde ASCOM compatibility with Software Bisque products. Please use the X2 drivers for TheSky X, and be sure you have TheSky X 10.1.11 or later. For more information see Working with TheSky (section 2).

Mount/Telescope X2 to ASCOM Converter for TheSky X (1.0.1 Jan 2012)

Plug-in that allows Software Bisque's TheSky X to control telescopes via ASCOM telescope drivers. Requires TheSky X Professional Edition for Windows 10.1.11 or later. Download then run this to install the plugin. Thanks to Bob Denny of DC-3 Dreams for this.

Focuser X2 to ASCOM Converter for TheSky X
Plug-in that allows Software Bisque's TheSky X to control focusers via ASCOM focuser drivers. Requires TheSky X Professional Edition for Windows 10.1.11 or later. Download then run this to install the plugin. Thanks to Evan Warkentine of ShrimpBox Astronomy for this.

CCDSoft™ ASCOM Camera Plug-In for CCDSoft
Use the X2 plug-ins above for TheSky X. Plug-in that allows Software Bisque's CCDSoft Imaging Software to control cameras via their ASCOM drivers. Useful for one shot color and other cameras not natively supported. Download then run this to install the plugin. Thanks to Evan Warkentine of ShrimpBox Astronomy for this.

SkyMap Pro ASCOM Plug-In
Plug-in that allows Chris Marriott's SkyMap Pro 11 planetarium and star-charting software to control telescopes via ASCOM telescope drivers. This is a zip file with a read-me document describing installation and usage.
