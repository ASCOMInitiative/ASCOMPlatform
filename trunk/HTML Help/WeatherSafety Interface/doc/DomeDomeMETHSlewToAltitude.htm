Dome.SlewToAltitude() Method

   

Slew the dome to the given altitude position.

Syntax

Dome.SlewToAltitude(Altitude)

The method syntax has these parts:

Part Description
Altitude (Double) Target dome altitude (degrees, horizon zero and increasing positive to 90 zenith)
Return (Nothing) Does not return a value.

Remarks

Raises an error if Dome.Slaved is True, if not supported, if a communications failure occurs, or if the dome can not reach indicated altitude.
