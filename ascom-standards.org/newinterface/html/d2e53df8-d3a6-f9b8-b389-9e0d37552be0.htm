Remarks

This is an optional property and can throw a PropertyNotImplementedException.

When Offset is operating in OFFSET VALUE mode:
  • OffsetMax must return the camera's highest valid Offset setting.
  • OffsetMax must be equal to or greater than OffsetMin.
  • Offsets must throw a PropertyNotImplementedException

Please note that OffsetMin and OffsetMax act together and that either both must be implemented or both must throw PropertyNotImplementedExceptions.

It is recommended that this function be called only after a connection is established with the camera hardware to ensure that the driver is aware of the capabilities of the specific camera model.

This property is only available in Camera Interface Version 2 and later.
