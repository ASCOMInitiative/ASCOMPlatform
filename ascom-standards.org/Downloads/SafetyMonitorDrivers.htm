Safety Monitor Drivers

SPECIAL COMPONENT - Safety Monitor Hub (June 2019)
SafetyMonitor driver for use with programs that may want to monitor more than one SafetyMonitor source (e.g. power conditions and weather). For more information click the button to visit the author's web site and download the driver.
Boltwood Safety Monitor (V6.2 January 2016)
This multi-object driver implements Safety Monitor drivers that interface to the Boltwood Weather station and it's clones. Two drivers are provided, OK to Image and OK to Open, they can have different safety limits. In addition, this driver also implements an Observing Conditions driver. This was developed by Chris Rowland. Please address messages relating to this on the ASCOM-Talk Users Group. Requires current ASCOM Platform.
Environment Safety Monitor (Jan 2021)
The Environment SafetyMonitor reads observations from an ObservationConditions Driver (preferably OCH, but any driver can be used) and allows to trigger SAFE/UNSAFE events on 11 observation types in a maximum of 13 rules (the 2 additional ones are to allow for upper/lower limits for wind direction and temperature). For more information click the button to visit the author's web site and download the driver.
Generic File Safety Monitor (Jan 2021)
The Generic File SafetyMonitor monitors any ASCII-file(s) on the local hard drive or on the network, scans it for a preamble (e.g. ROOF) and a trigger in that same line (e.g. CLOSED) and sets the SAFE/UNSAFE event based on the user settings. For more information click the button to visit the author's web site and download the driver.
Generic Power Condition Driver (June 2019)
Generic SafetyMonitor driver for use with programs such as SGP that have power monitoring and power failure capabilities. It works with the events in the Windows Application and System Events Logbook to provide safety monitoring with most any brand of UPS that lists its events in these Logbooks. Has passed the Conform tests. For more information click the button to visit the author's web site and download the driver. Lunatico's Combined Safety Monitor and Observing Conditions Driver
This combined driver (safety monitor and observing conditions) will work with Lunatico's range of CloudWatcher products, including the standard CloudWatcher, the pocketCloudWatcher and the Solo. It has been optimized to withstand short network problems. This also contains an Observing Conditions driver
PowerStatus Safety Monitor (June 2019)

This SafetyMonitor driver alternative uses the .NET SystemInformation.PowerStatus object to check for the power state and/or battery state of a laptop and uses that to set the IsSafe property.
