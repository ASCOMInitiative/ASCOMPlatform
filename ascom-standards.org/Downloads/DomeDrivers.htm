As of August 2022, we no longer host drivers here. Drivers formerly hosted here have been moved to the ASCOM Ghost & Legacy Drivers repository on GitHub.

Dome and Roof Drivers

Digital DomeWorks (external link)
Driver for the Technical Innovations Digital DomeWorks dome controller.
DPP Lesve Dome Net (external link)
Driver for the Lesve Dome Net control system. This is a rejuvenated effort for the original Lesve Dome, with new electronics and a .NET based driver. For more information visit the LesveDomeNet web site.
Lunatico Dragonfly Observatory Control
The software suite for the Dragonflyobservatory control add-on; remote control software and ASCOM dome and switch driver for fully automated roof and observatory operation.
MaxDome (external link)
Driver for the MaxDome II Dome Control System by Diffraction Limited. The latest driver for the older MaxDome I controller is also available here. Note that a firmware upgrade may be required for the latest driver!
NexDome (external link)
Driver for the NexDome dome controller.
Omega Lab OCS III Dome Driver (external link)
Omega Lab's OCS III includes an ASCOM Dome driver interface. For more information visit the Omega Lab OCS III web site.
ScopeDome Dome and Roll-Off Roof Observatory
The driver and control program for the ScopeDome dome and roll-off roof controller.
