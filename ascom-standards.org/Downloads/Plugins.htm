Plug-Ins

CCDSoft™ ASCOM Camera Plug-In for CCDSoft
Do not use this with TheSky X. Plug-in that allows Software Bisque's CCDSoft Imaging Software to control cameras via their ASCOM drivers. Useful for one shot color and other cameras not natively supported. Download then run this to install the plugin.
TheSky™ ASCOM Telescope (TeleAPI) Plug-In for TheSky 5/6 (5.0.4)
Do not use this with TheSky X. Plug-in that allows Software Bisque's TheSky 6/5 Astronomy Software to control telescopes via ASCOM telescope drivers. Download then run this to install the plugin. Use at your own risk, we do not support TheSKy 6.
AutomaDome™ ASCOM Dome (DomeAPI) Plug-In for AutomaDome
Do not use this with TheSky X. Plug-in that allows Software Bisque's AutomaDome Dome Control Software to control domes via ASCOM dome drivers. Download then run this to install the plugin.
CCDSoft™ ASCOM Focuser (FocusAPI) Plug-In for CCDSoft
Do not use this with TheSky X. Plug-in that allows Software Bisque's CCDSoft Imaging Software to control focusers via ASCOM dome drivers. Download then run this to install the plugin.
SkyMap Pro ASCOM Plug-In
Plug-in that allows Chris Marriott's SkyMap Pro 11 planetarium and star-charting software to control telescopes via ASCOM telescope drivers. This is a zip file with a read-me document describing installation and usage.
