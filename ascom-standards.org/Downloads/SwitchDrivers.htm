Switch Drivers

Digital Loggers, Inc. Web Power Switch (2019) for current models of this switch

This driver implements the ASCOM Switch interface, providing standard (and simple) calls to control the outlets on a late-model Digital Loggers Web Power Switch Pro. This updated version by Manoj Koushik uses the REST API provided by newer versions of the switch.

Please note

  • You must update your switch to the latest firmware!
  • In the controller web page
    • Set External APIs -> Allow REST-Style API: Enabled
    • Setup -> Access Settings -> Allow legacy plaintext login methods: Enabled

This package also includes some sample integration files for a popular remote astronomy package. Please address messages relating to this driver on the ASCOM Talk Users Group.

Digital Loggers, Inc. Web Power Switch (6.0.1) for older models of this switch
This driver implements the ASCOM Switch interface, providing standard (and simple) calls to control the outlets on a Digital Loggers Web Power Switch Pro. This version (originally done by Chris Rowland and Todd Benko) is no longer being maintained and should be used only if your switch is so old that the 2019 version above will not work with it. Please address messages relating to this driver on the ASCOM Talk Users Group.
LynxAstro Dew Controller

PegasusAstro Ultimate Powerbox
Download page for the PegasusAstro Ultimate Powerbox power control unit. Has an ASCOM Switch driver for outlet control, and other devices as well.
Seletek Dragonfly Observatory Control
The software suite for the Seletek Dragonfly (formerly Firefly) observatory control add-on; remote control software and ASCOM dome and switch driver for fully automated roof and observatory operation. Has passed switch conformance test.
