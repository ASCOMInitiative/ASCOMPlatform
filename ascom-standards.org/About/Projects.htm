Projects in Progress

ASCOM Platform 6

The next generation of the ASCOM Platform provides compatibility with Windows 7 User Account Control and 64-bit systems, some new device interfaces, and even more support for both drivers and astronomy applications. It contains new simulators, updated Visual Studio driver templates, and much more. This huge project is nearing completion with several people working on it, and is expected to be available in May 2011.

An absolute requirement for Platform 6 is that it be 100% compatible with existing astronomy software and drivers. It is currently being tested by driver and application developerws to make sure this requirement is met. For details on Platform 6 click the link below.

Detailed Description of ASCOM Platform 6 (April 2011)

Changes Coming in Platform 6 (for Developers)

  • Elimination of the requirement for early-binding interfaces in drivers. This will greatly ease the driver development process.
  • Return to registry based storage for Chooser registration and driver data storage. Again this will be transparent to drivers which use the Chooser and Profile objects.
  • Full support for 64-bit drivers, new simulators can run with 64-bit clients.

Integration with X2 and TheSky X

Software Bisque has implemented a cross-platform driver architecture called X2 that provides interfacing with asronomical devices from TheSky X Pro. The ASCOM X2 Integration project has the goal of developing generic X2 drivers that convert from X2 to ASCOM, allowing TheSky X to take advantage of all existing ASCOM drivers. The new X2 drivers will replace the old TeleAPI, FocusAPI, and DomeAPI plugins that are legacy from TheSky 6. The X2 interfaces provide a richer set of capabilities, allowing TheSky X to take advantage of more features of devices via their ASCOM drivers. In development are generic X2 to ASCOM drivers for Mounts, Focusers, Cameras, FilterWheels, Rotators, and Domes. Some of the X2 interfaces themselves are evolving, so we're tracking these developments. Look for the first set of these X2 to ASCOM drivers early Summer 2011.

Note that ASCOM is Windows-only, so these X2 to ASCOM drivers will not work with TheSky X Pro running on Mac OS X.
