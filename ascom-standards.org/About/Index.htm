What is the ASCOM Initiative?

The ASCOM Initiative is a loosely-knit group of developers and astronomical instrument makers that work together to bring vendor-independent and language-independent plug-and play compatibility between astronomy software and astronomical instruments on Windows, and now Linux and MacOS computers. ASCOM stands for the Astronomy Common Object Model.

If you are an astronomer and have problems with a device or application that uese ASCOM universal connectivity standards, please refer to the Support section for a roadmap for getting help

See the pages listed on the left for more information about ASCOM itself. ASCOM is not a program or software component, it is a set of universal interfacing standards, and a set of tools and specifications aimed at developers who wish to have their devices and apps operate in our universal ecosystem. See the Frequently Asked Questions section for more info.
